= JeePlcBus

== Description
Plugin permettant d’utiliser le module PLCBUS-1141 avec Jeedom. Ce plugin fonctionne en local. Une MAJ est en cours pour la version déportée (mode esclave).

Le plugin PlcBus permet de dialoguer avec l’ensemble des périphériques compatibles avec le module PLCBUS-1141.
Le module PLCBUS-1141 permet d’interfacer un ordinateur avec le signal PLCBUS afin de pouvoir contrôler les différents modules de l’installation à partir de celui-ci.
